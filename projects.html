  • A* Algorithm Visualizer

The program will find the quickest route between two clicked nodes using the A-Star algorithm, given that a path between the two exists.

  • Minimax Alphabeta Tic-Tac-Toe

    This project was for Dr. Julio Bahamon's Intro to Artificial Intelligence class. In it, I built an unbeatable Tic-Tac-Toe AI using adversarial search techniques. The computer evaluates all possible game outcomes to make the optimal move assuming both players play perfectly by exploring the game tree, and assigning scores to terminal states (win/loss/draw), (1/-1/0). There is an included option to prune branches that won't affect the final decision using Alpha-Beta pruning (a standard optimization to minimax) to reduce search time.
